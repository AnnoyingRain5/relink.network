Features

General:

  • Open chat platform
  • Server-to-Server federation
  • Infinite number of chatrooms
  • Username/Password authentication
  • Direct Messages
  • Notifications when mentioned in other channels
  • All official components are open source

Official Client:

  • Fully command-line based

Official Server:

  • Can be deployed via Docker
  • Configured via environment variables

Lightquark Bridge:

  • Connect a Lightquark channel to a Relink channel!
  • Supports two-way messaging
  • Configured via environment variables
  • Can be deployed via Docker
