Relink Client Setup

Downloading

Head over to the download page and download the latest version of the client software for your operating system. You can also download the python script directly from github if you prefer, but make sure to recursively clone it and pip install the requirements if you go that route!

Important note:

If you use Linux and are using a binary from the Downloads page, note that you will have to mark it as executable by using chmod +x filename

Starting the client

Important note:

If you are using Windows, you will need to ensure you are using a compatible console. The standard Command Prompt and Powershell terminals provided by Windows are not suitable due to a lack of support for many standardised escape codes. Windows Terminal (Microsoft Store link) does support these codes, and is the recommended terminal app for end users.

If you are using an executable from the download page, you will also need to set it as your default terminal app.

Windows:

If it's an executable from the download page, just double click it to open! If it's a script file, you'll need to open it from your terminal by running python3 client.py

MacOS and Linux

Open your chosen terminal app and run the app using it. This is often as easy as dragging the executable into the windows and hitting enter.

Connecting to a server

Each time you start the client you will be asked to input a server address.

If you do not know what server to join, feel free to join the official server at relink.network. Know that due to federation, which server you join does not really matter, as, just like email, you can join any channel and DM any user from any server.
