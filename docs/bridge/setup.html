Lightquark <--> Relink Bridge Setup

Downloading

Head over to the download page and download the latest version of the bridge software for your operating system. You can also download the python script directly from github if you prefer, but make sure to recursively clone it and pip install the requirements if you go that route!

There is also a dockerfile provided, which will automatically pull from git master each time you restart it.

Important note:

If you use Linux and are using a binary from the Downloads page, note that you will have to mark it as executable by using chmod +x filename

Configuration

Now that you have the software downloaded, it's time to do some configuration!

Below is the current git master version of the .env.example file, you can use this to either make your own .env file, or to set the environment variables yourself. Note that this default configuration will not work as-is, due to not having a valid token.

Well, how do I get a token then?

You can get a Lightquark token using the Lightquark API, which is documented on the Lightquark website.

Wait, how do I set environment variables anyway?

Please check our dedicated page on setting environment variables

Starting the bridge

Now that you have configured everything, you can now start the bridge! This is as easy as starting the script through the command line!
